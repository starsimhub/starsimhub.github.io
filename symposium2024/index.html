Starsim Learning Day

Oct. 3, 2024 | Seattle, USA | Satellite to the IDM Symposium

Background

We are pleased to announce a one-day training session on Starsim as a satellite session following the 2024 IDM Annual Symposium.

Compared to other modeling approaches, agent-based models (ABMs) allow for detailed simulation of specific scenarios, and enable comparison with a broader range of data. Over the last several years, we developed a suite of ABMs that have seen widespread use for modeling different diseases, including COVID-19 (Covasim), human papillomavirus (HPVsim), and family planning (FPsim). Recently, we codified the principles and philosophies common across these models into a standalone tool called "Starsim". Starsim is capable of simulating multiple co-transmitting pathogens across diverse networks, and is adaptable to a wide range of policy questions and data. Since 2020, we have trained over 200 people to use Starsim models. The overall purpose of these sessions will be to provide an overview of this work, as well as hands-on training.

Aims

  1. To platform and promote the analyses that have been done using Starsim, with the aim of fostering and expanding the user community;
  2. To provide details on the model design, architecture, and feature roadmap;
  3. To facilitate ‘hands-on’ sessions for people who are interested in using Starsim to develop a model, analyze data, or explore policy questions.

Who should attend

  • Potential users that want to see what Starsim can be used for to get ideas for how they might be able to use it in the future, without necessarily wanting to use it now
  • Experienced modelers that want to learn more about the interesting implementation details in Starsim but don't necessarily want to actually use Starsim
  • Modelers of varying experience levels that want to build models with Starsim in the short term

Outcomes

  1. Participants should have an understanding of the current and planned features in Starsim, and how these might relate to their own projects
  2. Participants should know how to go about starting to use Starsim (where to find the code, who to contact for assistance)
  3. Participants that followed the tutorials should have a working Starsim environment set up on their computer and be able to run simple simulations using modules shipped with Starsim
